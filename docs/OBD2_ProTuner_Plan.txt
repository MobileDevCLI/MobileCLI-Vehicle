TECHNICAL NOTES:
================
- MCU heartbeat: 04 71 01 00 89 (MIL=ON, DTCs=9)
- 0x08 frames = raw CAN relay, NOT decoded OBD2 params
- CAN decoder exists: LuZhengCanParseToyotaFJ (in canbus2 APK)
- MCU ignores all commands without CAN service handshake
- Build chain: aapt2 + javac(--release 8) + d8 + zipalign + apksigner
- Android SDK: ~/android-sdk/platforms/android-28/android.jar
